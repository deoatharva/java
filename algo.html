﻿Algorithm (Bubble Sort) BUBBLE(LA, N)
Here LA is an array with N elements. This algorithm sorts the elements in LA.
1. Repeat Steps 2 and 3 for K = 1 to N - 1. 
2. Set PTR := 1. [Initializes pass pointer PTR.] 
3. Repeat while PTR ≤ N - K: [Executes pass.]
	(a) If LA[PTR] < LA[PTR + 1], then:
		Interchange LA[PTR] and LA[PTR + 1]. 
	    [End of If structure.]
	(b) Set PTR := PTR + 1.
	[End of inner loop.]
	[End of Step 1 outer loop.]
4. Exit.
